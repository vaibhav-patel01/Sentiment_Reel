> ⚠️ This extension is not published on the Chrome Web Store due to its 500 rs charge. Follow the steps below to install it manually in developer mode.

## To use this extension, these are the steps:

1. **Download the frontend**
   - Click the green `Code` button on this repository and select `Download ZIP`, or clone it:
     ```bash
     git clone <repo-url>
     ```
   - Extract the ZIP file if you downloaded it (skip this if you cloned).

2. **Open Chrome Extensions page**
   - Open Google Chrome.
   - Go to `chrome://extensions` in the address bar.

3. **Enable Developer Mode**
   - Toggle on **Developer mode** in the top-right corner of the Extensions page.

4. **Load the extension**
   - Click **Load unpacked**.
   - Select the folder where you downloaded/extracted the extension (the folder that contains `manifest.json`).

5. **Pin and use the extension**
   - Click the puzzle-piece icon in the Chrome toolbar and pin the extension for easy access.
   - Click the extension icon to start using it.

## Notes

- Since this extension is loaded unpacked (not from the Web Store), Chrome may show a "Developer mode extensions" warning banner — this is expected and safe to ignore for local use.
- If you make changes to the code, go back to `chrome://extensions` and click the **Reload** icon on the extension card to apply updates.
- To remove the extension, click **Remove** on the extension card at `chrome://extensions`.
